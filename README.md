## idea配置热部署
````
1,在pom中添加devtools的依赖
2,在settings中build->compiler中勾选auto build project
3,开启idea热部署设置，快捷键[ctrl+shift+alt+?]->regist..->compiler.automake.allow.when.app.running
````
